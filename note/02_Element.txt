1. Element

    - Component에 props를 전달하여 생성하는 결과물
    - 객체지향의 인스턴스 개념과 비슷하다 
    - Element는 일단 생성되고 나면 변경이 불가능하다 (immutable, 불변성의 법칙)
    - 생성된 Element의 props를 수정할 수 없기 때문에
      Element를 변경하고 싶은 경우에는 새 Element를 생성하여 교체해야 한다

2. 렌더링

    - 등록된 컴포넌트를 화면에 그리는 과정
    - Virtual DOM의 내용이 변경된 것 만으로는 화면에 변경 사항이 반영되지 않는다
    - 필요할 때만 렌더링을 호출하는 방식으로 화면 처리 자원의 낭비를 줄일 수 있다
