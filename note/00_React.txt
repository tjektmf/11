00_React 설치하기.txt

1. 설치 순서

 (1) VS Code 설치
 (2) Node.js 설치하기
 (3) Ctrl + Shift + ` 로 터미널을 연 후 
 (4) npm install npm -g
 (5) npx create-react-app [프로젝트명]
 (6) cd [프로젝트폴더명]
 (7) 프로젝트 위치에서 npm start
 (8) Ctrl + C 해서 일괄종료 가능(서버종료)

2. React란 무엇인가

    - 웹 페이지를 좀 더 체계적으로 개발할 수 있는 자바스크립트 라이브러리
    - HTML/CSS/Javascript 를 사용하면 유지보수가 힘들어지는 문제를 해결하기 위해 만들어진 라이브러리
    - 페이스북을 개발할 때 사용했던 라이브러리가 오픈 소스로 공개된 것이다

3. React의 특징

    - 마크형이 밀어주는 언어라서 안정감이 높고 커뮤니티가 활성화되어 있음
    - 계속 바뀌기 때문에 끊임없이 공부해야 함
    - 컴포넌트 단위로 웹 페이지를 조립하듯이 구성할 수 있음
    - SPA(Single Page Application)를 개발하기에 최적화되어 있는 라이브러리
    - 비교적 적은 비용으로 DOM을 교체할 수 있게 실제 DOM이 아닌 가상 DOM을 사용함
