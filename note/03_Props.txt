1. Props

    - 해당 컴포넌트의 속성을 의미하는 객체
    - 동일한 컴포넌트로 각자의 속성값(props)을 다르게 하여 엘리먼트를 생성할 수 있다
    - 엘리먼트 생성 후에는 props 값을 변경할 수 없다 (불변성)
    - 리액트 컴포넌트를 설계할 때 같은 props를 넣는다면 항상 동일한 element가 생성되도록 설계해야 한다 (함수의 순수성)
    - JSX를 사용할 때는 HTML 속성처럼 보기 좋게 전달할 수 있다
    - React.createElement()를 사용할 때는 두 번째 인자값에 객체 형태( {ㅇㅅㅇ} )로 전달해야 한다
